# GradePulse

Real-time grade & attendance tracker with instant WhatsApp alerts to parents.

## Features (Planned — Will be implemented)

- Teacher login (email + password)
- Excel upload → auto parse student grades (40 students in < 2 seconds)
- Instant WhatsApp alert to parents when grades are uploaded
- Daily attendance marking (Present/Absent/Late)
- Automatic WhatsApp attendance report to parents every morning
- Low-grade alert (< 40%) with red flag
- Student dashboard: GPA trend chart (last 5 exams)
- Syllabus progress tracker (chapter-wise %)
- Event & PTM reminders via WhatsApp
- Works on JioPhone (no app install needed — pure WhatsApp + browser)

## Tech Stack

- Spring Boot 3.4.11 (Java 17)
- MySQL + Flyway migrations
- Thymeleaf + Bootstrap 5
- Twilio WhatsApp API
- Apache POI (Excel parsing)
- JWT + Spring Security
- Hosted on Railway.app (free tier)
